My work spans both frontend and backend engineering—from build tooling, developer experience, and distributed frontend architectures to backend services and platform infrastructure.

I work across Java, Spring Boot, MongoDB, TypeScript, React, webpack, and Module Federation. I'm particularly interested in understanding how complex software systems work under the hood, identifying performance bottlenecks, and building solutions that improve reliability, scalability, and developer productivity.
